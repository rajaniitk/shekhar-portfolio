# AI-Powered EDA & Feature Engineering Platform

A comprehensive web application for exploratory data analysis, statistical testing, and feature engineering with AI-powered insights.

## Features

### 🔍 Data Analysis
- **40+ Statistical Tests**: Normality, hypothesis testing, correlation analysis, independence tests, and more
- **Comprehensive EDA**: Automated exploratory data analysis with detailed insights
- **Advanced Outlier Detection**: Multiple methods including IQR, Z-score, Isolation Forest
- **Data Quality Assessment**: Completeness, consistency, and validity checks

### 📊 Visualization
- **60+ Chart Types**: From basic histograms to advanced 3D plots, radar charts, and treemaps
- **Interactive Visualizations**: Powered by Plotly for dynamic, responsive charts
- **Customizable Themes**: Dark and light themes with professional styling
- **Export Capabilities**: Download charts in PNG, SVG, and PDF formats

### 🛠️ Feature Engineering
- **Numerical Transformations**: Scaling, mathematical transformations, binning
- **Categorical Encoding**: One-hot, label, frequency, and target encoding
- **DateTime Features**: Extract temporal components with cyclical encoding
- **Text Processing**: Character counts, sentiment analysis, TF-IDF
- **Advanced Features**: Polynomial features, interactions, PCA, clustering

### 🧪 Statistical Testing
- **Normality Tests**: Shapiro-Wilk, Anderson-Darling, Kolmogorov-Smirnov, Jarque-Bera
- **Hypothesis Tests**: t-tests, ANOVA, Mann-Whitney U, Wilcoxon, Kruskal-Wallis
- **Correlation Tests**: Pearson, Spearman, Kendall's tau, partial correlation
- **Independence Tests**: Chi-square, Fisher's exact test, G-test
- **Variance Tests**: Levene's, Bartlett's, Fligner-Killeen tests

### 💡 AI Insights
- **Automated Recommendations**: Data quality suggestions and transformation tips
- **Natural Language Interpretations**: Plain English explanations of test results
- **Smart Feature Selection**: AI-guided feature importance and selection
- **Data Patterns**: Automatic detection of relationships and anomalies
